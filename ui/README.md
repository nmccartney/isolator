# essentials

## Project setup

```
# npm
npm install

```

### Compiles and hot-reloads for development

```
# npm
npm run dev
```

### Compiles and minifies for production

```
# npm
npm run build
```

### Lints and fixes files

```
# npm
npm run lint
```

### Run with Android Studio

```
npx cap sync

npx cap run android
# OR
npx cap open android
```
